# Changelog

## 3.0.0 · Acceso público directo

- Eliminada la pantalla de acceso.
- Eliminadas la clave, la sesión local y las redirecciones.
- Eliminados `login.*`, `access-config.js` y el configurador de clave.
- Conservada la carga de recursos desde el Release `assetsipath`.
- Conservadas medidas visuales disuasorias y bloqueo de impresión.
- Actualizados validadores y documentación.

## 6.1 · GitHub Pages + Release

- Carga directa de imágenes y vídeo desde el Release `assetsipath`.
- Resolución automática del propietario y repositorio desde la URL de GitHub Pages.
- Compatibilidad con un repositorio público independiente de recursos.
- Eliminada la dependencia de `assets/`, Cloudflare Functions y secretos de servidor.
- Conservada la pantalla de acceso disuasoria y la marca de sesión.
