# Catálogo iPath definitivo · v5.5

Versión adaptada a la estructura definitiva de assets aportada el 23/07/2026.

## Instalación

Copia `index.html`, `styles.css`, `app.js`, `asset-manifest.json` y `validar-assets.ps1` en la raíz de `Catálogo iPath definitivo`, conservando la carpeta `assets` existente.

La versión usa directamente:

- la nueva portada;
- todas las galerías por producto;
- las carpetas Vivienda Pública, Colegios y Equipamiento sanitario y Ciudadano;
- las tres imágenes de proceso;
- los iconos 3D para capacidades digitales y operativas;
- el vídeo MP4 de Plazas y espacios cívicos;
- la lámina única de texturas;
- el GIS oscuro de Aspe.

Ejecuta `validar-assets.ps1` para comprobar nombres y rutas exactos.

Los recursos de `assets/generated` se mantienen solo como respaldo si una imagen principal no está disponible.
