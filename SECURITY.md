# Política de seguridad y publicación

## Alcance

Esta edición es pública y estática. No contiene autenticación ni secretos.

No deben incorporarse al repositorio contraseñas, tokens personales de GitHub, claves de API, datos personales, documentación interna de la pila tecnológica o información contractual no destinada al catálogo.

## Recursos del Release

No añadas tokens a `release-config.js` ni a ningún JavaScript. El Release debe ser público para que GitHub Pages pueda cargar sus adjuntos.

## Protección visual

La marca de agua, el bloqueo de impresión y las restricciones de interacción son medidas disuasorias. Una web pública no puede impedir completamente capturas, fotografías externas, inspección técnica del código o descarga de un recurso público cuya URL sea conocida.

## Derechos

La publicación pública no concede derechos de copia, modificación, redistribución ni explotación comercial. Consulta `LICENSE.md`.
